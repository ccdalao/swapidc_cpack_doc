# SWAPIDC 共存版文档



## ①前言

何为共存版？

下面用一个简单的式子表达

共存版 = SWAPIDC官方版 + 第三方版

在保留SWAPIDC官方生态的情况下，进行升级

*作为官方版的分支进行发行，保证您可以在共存版和SWAPIDC官方版之间**自由选择，还原，安装***
